# Hackathon 2026 Agent

This is an agentic workflow built using the Google ADK (Agent Development Kit). 

## Setup

1. Create a virtual environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -e .
   ```

3. Configure environment variables in `.env` (copy from `.env.template` if needed).

## Running

Run the agent locally:
```bash
hackathon2026_agent
```
